# personal-site

Source for [martingranados.com](https://martingranados.com). Astro + TypeScript (strict) + hand-rolled CSS + MDX, hosted on Cloudflare Pages.

See [AGENTS.md](./AGENTS.md) for project conventions, scripts, and hard rules.

```sh
pnpm install
pnpm dev      # dev server at http://localhost:4321
pnpm verify   # typecheck + lint + build + test (the gate)
```
